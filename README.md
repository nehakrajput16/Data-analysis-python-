# Data Analysis using Python

## Description
This project focuses on performing basic data analysis using Python.
The goal of the project is to understand how raw data can be cleaned,
analyzed, and visualized to extract useful insights.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib

## Features
- Load and explore dataset
- Perform basic statistical analysis
- Handle missing or inconsistent data
- Visualize data using graphs

## Learning Outcome
- Gained hands-on experience with data analysis libraries
- Improved understanding of data preprocessing and visualization
- Learned how data-driven insights are generated
